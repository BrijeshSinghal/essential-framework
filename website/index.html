Essential: A dynamic, barebones CSS framework

The aim is to allow the developer to be able to enter whatever property value/measurement he wants, as classes. For example,

m-300px

translates to

margin: 300px

Extremely light weight

Contains only essential CSS properties. The goal here is to give easy access to basic utilities as classes. While it is possible to design an entire website using essential only, it is not intended to serve as your own custom stylesheet replacement.

A complete list of availaible classes can be found on the documentation page.

Familiar Bootstrap Terminology

I have tried to make almost every class similar to/same as it's counter-part in Bootstrap. What this framework does not contain is all the clutter that comes with Bootstrap which you do not need if you want to build something quick, while still giving the developer access to complete tools for customisation.

Work in progress

I created this for my own personal use to be able to create a layout for a project quickly and then use custom styling to add details. I have uploaded the complete source to GitHub. This is written in SCSS and JS.
